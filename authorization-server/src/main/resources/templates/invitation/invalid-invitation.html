Invalid Invitation

The invitation link you're trying to use is not valid.

Error message

This might happen because:

  • The invitation link has expired
  • The invitation has already been used
  • The invitation was cancelled by an administrator
  • The link was copied incorrectly

What you can do:

  • Contact the person who invited you for a new invitation
  • Check your email for the most recent invitation
  • Make sure you're using the complete invitation URL
